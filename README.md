# README

## Setup

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed

### Steps
1. Open PowerShell
2. Run `docker pull tatchapero/talent_api:latest`
3. Start container: `docker run -d -p 8080:8080 --name talent_api tatchapero/talent_api`
3. Open browser at http://localhost:8080/swagger-ui/index.html
